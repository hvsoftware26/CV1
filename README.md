# CÀI ĐẶT MÔI TRƯỜNG
## Cài đặt ban đầu
### 1. Tạo venv( môi trường ảo)
1. Cài đặt Python 10.x trên máy 
2. Tại dự án, tạo venv bằng lệnh **`py -m venv .venv`**
3. Kích hoạt môi trường ảo 
-Chạy trên CMD VSCode `.venv\Scripts\activate.ps1`
-Chạy trên CMD windows `.venv\Scripts\activate.bat`
4. Cài đặt thư viện vào dự án `pip install -r requirements.txt`
